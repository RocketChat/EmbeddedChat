Embedded Chat

An easy-to-use, full-stack component (React.js + backend behaviors) for embedding Rocket.Chat into your web app.

Easy to Use

Embedded was developed to easily integrate Rocket.Chat with your React Application.

SDK Driven UI

The component uses the Rocket.Chat SDK for tight integration and features a fully custom-built UI, ensuring a cohesive and integrated user experience.

Customization

Tailor the chat window's appearance and functionality with pre-built designs, login options, and extensibility features.
